FitTrack is a full-stack web application for tracking workouts and physical activities.  
It allows users to register, log in, manage their workouts, view statistics and calendar events, while administrators have additional privileges for managing users and all workouts.


## Technologies Used
### Frontend
- React

### Backend
- Node.js
- Express.js
- SQLite
- JWT (JSON Web Token)
- bcrypt


## Features

### Authentication & Security
- User registration and login
- Password hashing with bcrypt
- JWT-based authentication
- Protected API routes
- Role-based authorization (user / admin)

### User Features
- Add, edit and delete workouts
- View workouts in a calendar
- Filter and sort workouts
- Weekly statistics (number of workouts, total duration, most active day)
- Support for different activity types (including gym exercises)

### Admin Features
- View all registered users
- Change user roles (user / admin)
- View all workouts from all users
- Delete any workout as admin
